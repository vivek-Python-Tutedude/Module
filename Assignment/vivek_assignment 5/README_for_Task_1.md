# 📘 Task 1 - Create a Dictionary of Student Marks

## 📌 Problem Statement

Write a Python program that:

1. Creates a dictionary where student names are keys and their marks are values.
2. Asks the user to input a student's name.
3. Retrieves and displays the corresponding marks.
4. If the student's name is not found, displays an appropriate message.

---

## 🎯 Objective

The objective of this program is to learn how to use dictionaries in Python to store and retrieve data based on keys, and to handle cases where a key does not exist.

---

## 🛠️ Technologies Used

- Python 3.x

---

## 📚 Concepts Covered

- Dictionary
- Key-Value Pairs
- User Input (`input()`)
- Conditional Statements (`if-else`)
- Dictionary Membership Operator (`in`)

---

## ▶️ How to Run

1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Run the program using:

```bash
python Task_1.py
```

4. Enter the student's name when prompted.
5. The program will display the student's marks if found; otherwise, it will display **"Student not found"**.

---

## 💻 Sample Output

### Case 1: Student Found

```
Enter the student's Name: Vivek

Vivek's Marks = 98
```

### Case 2: Student Not Found

```
Enter the student's Name: john

Student not found
```

---

## 📖 Program Explanation

### Step 1

Create a dictionary containing student names as keys and their corresponding marks as values.

### Step 2

Accept the student's name as input from the user.

### Step 3

Check whether the entered name exists in the dictionary.

### Step 4

If the name exists, display the student's marks.

### Step 5

If the name does not exist, display **"Student not found"**.
