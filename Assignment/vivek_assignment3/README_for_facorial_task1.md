# Factorial of a Number Using Loop and Recursion

## 📌 Project Description
This Python program calculates the factorial of a user-entered number using two different methods:
1. Using a loop
2. Using recursion

## ⚙️ Program Features
- Takes a number as input from the user.
- Calculates the factorial using a `for` loop.
- Calculates the factorial using a recursive function.
- Displays the factorial value for both methods.

## 🧠 Concepts Used
- User Input
- Functions
- For Loop
- Recursion
- Conditional Statements
- Arithmetic Operations

## ▶️ How to Run
1. Open the program in VS Code.
2. Run the file using Python:

```bash
python factorial_task1.py
```

3. Enter a number when prompted.
4. View the factorial calculated using both loop and recursion.
