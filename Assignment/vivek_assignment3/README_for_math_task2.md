# Mathematical Operations Using Python Math Module

## 📌 Project Description
This Python program uses the built-in `math` module to perform different mathematical operations on a user-provided number.

## ⚙️ Program Features
- Takes a number as input from the user.
- Calculates the square root of the number.
- Calculates the natural logarithm of the number.
- Calculates the sine value of the number.
- Displays all the calculated results.

## 🧠 Concepts Used
- User Input
- Python Math Module
- Mathematical Functions
- Variables

## ▶️ How to Run
1. Open the program in VS Code.
2. Run the file using Python:

```bash
python math_module_task2.py
```

3. Enter a number when prompted.
4. View the square root, logarithm, and sine values.
