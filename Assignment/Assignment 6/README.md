# Tkinter Calculator

## Problem Statement

Create a Python calculator application using the Tkinter library that
provides a graphical user interface (GUI) for performing basic
arithmetic operations.

------------------------------------------------------------------------

## About the Project

This project is a simple calculator built using Python and Tkinter. It
allows the user to enter numbers using on-screen buttons and perform
basic mathematical operations. The result is displayed in the input
field.

------------------------------------------------------------------------

## Features

-   Graphical User Interface (GUI)
-   Number buttons (0--9)
-   Addition (+)
-   Subtraction (-)
-   Multiplication (\*)
-   Division (/)
-   Equal (=) button to display the result
-   Clear button to reset the input

------------------------------------------------------------------------

## Technologies Used

-   Python 3
-   Tkinter

------------------------------------------------------------------------

## File

    vivek_Calculator.py

------------------------------------------------------------------------

## How to Run

1.  Install Python 3.
2.  Open the project folder in VS Code or any Python IDE.
3.  Run the following command:

``` 
bash
python vivek_Calculator.py
```

------------------------------------------------------------------------

## Working

1.  Click the number buttons to enter the first number.
2.  Select the required operation.
3.  Enter the second number.
4.  Press **=** to see the result.
5.  Press **Clear** to clear the display.
